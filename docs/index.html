Overview

TypeAgent is a TypeScript library that explores an architecture for building personal agents with natural language interfaces using TypeChat. It provides a set of application agents that can take actions based on user intent, with actions defined using TypeChat schemas.

Installation

You can install TypeAgent using npm:

npm install @typeagent/core

Usage

Here's a basic example of how to use TypeAgent:

import { Agent } from '@typeagent/core';

// Initialize your agent
const agent = new Agent({
  // Configuration options
});

// Handle user input
agent.processRequest('Create a new task for tomorrow');
