# Workflow examples

A bundle of workflow definitions consumed by the engine, CLI, and adapter:

- `ir/` &mdash; Hand-written workflow IR (`*.json`). These are checked in and
  loaded as-is.
- `dsl/` &mdash; DSL source files (`*.wf`). Compiled by `wfc` to IR JSON.
- `built/` &mdash; Compiled output of `dsl/*.wf` (`*.json`). Generated by the
  package build; ignored from source control.

The `workflow-agent` adapter scans both `ir/` and `built/` at startup so that
hand-written and DSL-authored workflows appear together. Run the build to
refresh compiled output:

```
pnpm --filter workflow-examples build
```
