Grammar Tools UI - Dev Harness

All components below use the FixtureBackend with canned player-grammar data. No real grammar-tools-core or file system access is needed.

D.6: gt-debug-panel (composite)

Full debug panel with grammar picker, tab bar, and all sub-components.

D.1: gt-completion-panel

Standalone completion preview. Pre-filled with "play " to show live results.

D.2: gt-trace-timeline

Standalone trace view. Pre-filled with the fixture trace input.

D.3: gt-source-view (grammar picker)

Grammar source selector: file, agent, or live.

gt-rule-list

Navigable list of grammar rules.

D.4: gt-coverage-heatmap

Coverage report rendered from fixture data.

D.5: gt-diff-view

Side-by-side rule diff with fixture data (all changed rules expanded).
