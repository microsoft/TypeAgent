# IR Future Ideas

This folder tracks exploratory IR ideas that are not part of the current
contract.

Use one file per idea.

Suggested template:

- Title
- Status
- Context
- Candidate option(s)
- Why consider this
- Risks and costs
- Open questions
- Non-goal

When an idea is accepted, fold the final behavior into
`docs/design/workflowSystem/ir/ir-v0.2.md` (or the active IR spec), then mark
or remove the future note.
